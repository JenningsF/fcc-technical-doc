Intro

With the advent of mobile-first web development, many UI designs are beginning to include toggle buttons and switches. Unfortunately, there are no pre-built CSS elements that implement toggle buttons. The good news is we can use input checkboxes and radio buttons along with some styling to achieve the toggle buttons we desire. Let's get started!

Checkbox vs. Radio

When deciding what type of input type to use, either checkbox or radio, it is best to first decide what will your toggle button would be used for. Here are some considerations:

  • If your toggle button will be used as a simple binary on/off switch, a checkbox is recommended.
  • If your toggle button will be used to be more specific options (i.e. light/dark theme toggle), a radio button is recommended.

Checkbox Toggle Switch

The checkbox toggle switch typically does not include any inline labels and should either be self-explanatory or accompanied by labels near the button itself

  • HTML

    The HTML is simply an input tag with the checkbox type and a span tag with corresponding classes for both the input and span 

    <label class="checkbox-toggle-switch">
    <input type="checkbox" class="check-toggle">
    <span class="slider"></span>
</label>

  • CSS

    .checkbox-toggle-switch {
    position: relative;
    display: inline-block;
    /* width of toggle switch */
    width: 60px;
    /* height of toggle switch */
    height: 34px;
}

/* hides the actual checkbox but keeps it clickable */
.checkbox-toggle-switch input {
    opacity: 0;
    width: 0;
    height: 0;
}

.slider {
    position: absolute;
    cursor: pointer;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    /* color of switch when unchecked */
    background-color: #ccc;
    /* adjusts how long the switch transitions*/
    transition: 0.4s;
    /* should match height of toggle switch above */
    border-radius: 34px;
}

.slider:before {
    position: absolute;
    content: "";
    /* height and width of internal button */
    height: 26px;
    width: 26px;
    /* left and bottom should match */
    /* displacement of internal button */
    left: 4px;
    bottom: 4px;
    /* color of inside of switch */
    background-color: white;
    /* should match transition time set above */
    transition: 0.4s;
    /* creates round button */
    border-radius: 50%;
}

input:checked + .slider {
    /* color of switch when checked */
    background-color: #2196f3;
}

input:focus + .slider {
    /* shadow that appears behind button */
    box-shadow: 0 0 1px #2196f3;
}

input:checked + .slider:before {
    /* distance traveled when button is clicked */
    transform: translateX(26px);
}

Radio Toggle Switch

The radio toggle switch typically has labels build into them and does require additional labels near the button itself

  • HTML

                            
<span class="radio-toggle-switch">
    <input type="radio" class="radio-toggle" id="on" name="select" value="on" checked>
    <label class="radio-toggle-label" for="on">On</label>
    <input type="radio" class="radio-toggle" id="off" name="select" value="off">
    <label class="radio-toggle-label" for="off">Off</label>
</span>

  • CSS

    .radio-toggle-switch {
    font-size: 0;
    margin: 0 1rem;
}

.radio-toggle {
    position: absolute;
    clip: rect(0, 0, 0, 0);
    height: 1px;
    width: 1px;
    border: 0;
    overflow: hidden;
}

.radio-toggle-label {
    color: white;
    background-color: #ccc;
    display: inline-block;
    font-size: 1rem;
    font-weight: 400;
    line-height: 1;
    text-align: center;
    vertical-align: middle;
    padding: 0.5rem 1rem;
    border: 1px solid #ccc;
    border-radius: 9999px;
    transition: all 0.17s ease-in-out;
}

.radio-toggle-label:hover {
    cursor: pointer;
    color: white;
}

.radio-toggle:checked + .radio-toggle-label {
    color: white;
    background-color: #2196f3;
}

.radio-toggle-label:first-of-type {
    border-radius: 9999px 0 0 9999px;
    margin-right: 0rem;
}

.radio-toggle-label:last-of-type {
    border-radius: 0 9999px 9999px 0;
    margin-left: -2px;
}
